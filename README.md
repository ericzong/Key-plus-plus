# 快捷键

| Casplock+ | 功能                     | 说明     |
| --------- | ------------------------ | -------- |
| b         | Backspace                |          |
| d         | 删除当前行               |          |
| x         | Delete                   |          |
| h         | 左移                     |          |
| j         | 下移                     |          |
| k         | 上移                     |          |
| l         | 右移                     |          |
| u         | 左移一个“单词”           |          |
| p         | 右移一个“单词”           |          |
| y         | 转到行首                 | Home     |
| ;         | 转到行尾                 | End      |
| Shift+h   | 向左选择一个字符         |          |
| Shift+l   | 向右选择一个字符         |          |
| Shift+u   | 向左选择一个“单词”       |          |
| Shift+p   | 向右选择一个“单词”       |          |
| Shift+y   | 向左选择到行首           |          |
| Shift+;   | 向右选择到行尾           |          |
| Enter     | 任意位置换行             |          |
| ESC       | 退出脚本                 |          |
| Shift+,   | 当前选择文本两边加尖括号 | \<text\> |
| Shift+9   | 当前选择文本两边加圆括号 | (text)   |
| [         | 当前选择文本两边加方括号 | [text]   |
| Shift+[   | 当前选择文本两边加花括号 | {text}   |
| '         | 当前选择文本两边加单引号 | 'text'   |
| Shift+'   | 当前选择文本两边加双引号 | "text"   |
